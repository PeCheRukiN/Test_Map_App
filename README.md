# Test_Map_App
Для запуска необходимо:
1. Выполнить установку CocoaPods;
2. Импортировать #import <UIKit/UIKit.h> в Pods/Pods/YandexMapKit/YMKAnnotationImage.h 
3. Добавить в проект следующие фреймворки и библиотеки:

AudioToolbox

OpenAL

AVFoundation

CoreData

CoreLocation

CoreTelephony

libsqlite3.dylib

libstdc++

libxml2.dylib

libYandexMapKit

libz.dylib

MessageUI

OpenGLES

QuartzCore

Security

SystemConfiguration
